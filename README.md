# OmniMind – AI Platform for Coding, Creativity & Learning

</div>

OmniMind is an all-in-one AI platform designed to help users **create, learn, and code faster**. It combines AI-powered tools for real-time dialogue, video generation, knowledge summarization, and live tutoring into a single, powerful platform.

---

## Features

- **AI Dialogue** – Get instant answers with clear visual explanations.  
- **Video Forge** – Turn ideas into 1080p videos with just a prompt.  
- **Knowledge Stream** – Summarizes books, documents, and sessions automatically.  
- **Live Tutoring** – Personalized guidance with real-time feedback and progress tracking.  
- **Secure Sign-In** – Email/password or Google sign-in options.

## Run Locally

**Prerequisites:** Node.js  

#### 1. Install dependencies
npm install

#### 2. Set the GEMINI_API_KEY in .env.local to your Gemini API key
echo "GEMINI_API_KEY=your_api_key_here" >> .env.local

#### 3. Run the app
npm run dev

#### 4. Open your browser and navigate to
echo "Open http://localhost:3000 in your browser"
